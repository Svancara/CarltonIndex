    Here is a CSV datafile containing data the above www pages were generated from. Carlton_index_2020-11-06.zip

    That data are selected using this query: 

      select
   `date` as calculated,
   c.systemid,
   s.SystemName,
   DATEDIFF(`date`,date(from_unixtime(c2s.added))) as AgeDays,
   s.StartingCash,
   eq.totalpl as Equity,
   (select min(totalpl) from c2ex_equity_daily eq where eq.systemid = c.systemid) as MinEquity,
   ReturnAnnual,
   MaxDrawdown,
   NumTrades,
   ProformaReturnRolling180,
   ProformaReturnRolling210,
   ProformaReturnRolling240,
   ProformaReturnRolling270,
   LeverageWght180,
   LeverageWght210,
   LeverageWght240,
   LeverageWght270,
   LeverageWghtSMA7180,
   LeverageWghtSMA7210,
   LeverageWghtSMA7240,
   LeverageWghtSMA7270
 from carlton_score c
 join systems s on s.systemid=c.systemid
 join c2ex_equity_daily eq on eq.systemid=c.systemid and eq.datetime=c.date
 join c2systems c2s on c2s.guid=c.systemid
 where date > "2018" and 0.1 <= ReturnAnnual and ReturnAnnual <= 0.4 and MaxDrawdown < 0.6
 order by 1,2
