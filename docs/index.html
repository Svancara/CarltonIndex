Results from 2020-11-10 - Leverage data

  • Results are provided for the latest and some historical dates.
  • Each date contains pages for all rolling periods.
  • Each rolling period page contains columns for all three types of calculations: ProformaReturn, LeverageWeighted, LeverageWeighted smoothed using SMA(7) as specified on Slack.
  • Conditions for systems selection: 10% <= AnnReturn <= 40% and MaxDrawdown <= 60% (see tables captions )
  • Tables on the WWW pages are sortable, can be filtered and searched and columns can be reordered. Default sorting: ProformaReturnRolling DESC
  • All values are calculated as of the specified historical dates. Except the number of trades that is at the last date. (I made an error).
  • Strategies are preselected as before.
    • Live strategies only.
    • Strategies ages must be greater than 180 + 30 days (today(*)). (The shortest rolling period + 30 days for monthly returns.)
    • (*) Strategies ages are as of the latest date (today). It means some strategies does not have enough data in historical calculations.

All that is here.

Here is a CSV datafile containing data the above www pages were generated from. Carlton_index_2020-11-06.zip

That data are selected using this query: 

  select
   `date` as calculated,
   c.systemid,
   s.SystemName,
   DATEDIFF(`date`,date(from_unixtime(c2s.added))) as AgeDays,
   s.StartingCash,
   eq.totalpl as Equity,
   (select min(totalpl) from c2ex_equity_daily eq where eq.systemid = c.systemid) as MinEquity,
   ReturnAnnual,
   MaxDrawdown,
   NumTrades,
   ProformaReturnRolling180,
   ProformaReturnRolling210,
   ProformaReturnRolling240,
   ProformaReturnRolling270,
   LeverageWght180,
   LeverageWght210,
   LeverageWght240,
   LeverageWght270,
   LeverageWghtSMA7180,
   LeverageWghtSMA7210,
   LeverageWghtSMA7240,
   LeverageWghtSMA7270
 from carlton_score c
 join systems s on s.systemid=c.systemid
 join c2ex_equity_daily eq on eq.systemid=c.systemid and eq.datetime=c.date
 join c2systems c2s on c2s.guid=c.systemid
 where date > "2018" and 0.1 <= ReturnAnnual and ReturnAnnual <= 0.4 and MaxDrawdown < 0.6
 order by 1,2

Results from 2020-10-27

There are 8 pages there.

  • All rolling periods were generated: 180, 210, 240, 270 days
  • Each page is ordered by an appropriate colum (i.e. ProformaReturns210) descending.
  • Four pages are without any limitations. Another four pages are limited to systems having Annual return in the 0.1 - 0.5 range.
  • Equity pictures with rolling points lines added. It means: if the rolling interval is 210, a red vertical line is drawn 210 days back from the end of the chart.
  • Equity pictures are drawn from zero dollars on the Y axe. A horizontal line of the leftmost value is added. This line should represent a startig capital.
